三方登录(判断手机号是否存在)¶

三方登录
POST/user/preBind

三方登录流程：1。直接登录，判断openid是否已绑定。2。绑定手机号，判断手机号是否已经注册。3。如果注册输入密码进行绑定 4。如果没注册，则进行注册流程,注册时把openid传入

请求参数:

  • phone(required, string, 120) … 手机号

返回参数:

  • code (int) - 2000/4000

  • data (boolean) - 是否存在true/false
